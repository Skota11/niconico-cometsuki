# ニコニココメ付き保存

ニコニコ動画・ニコニコ生放送の映像を**コメント付きで録画・保存**できるChrome拡張機能です。

## 機能

- ニコニコ動画の動画をコメント付きで録画・保存
- ニコニコ生放送をリアルタイムでコメント付き録画
- ポップアップUIからワンクリックで録画開始/停止
- キーボードショートカット対応

## インストール方法

### 手動インストール

1. このリポジトリをダウンロード  
   → 右上の「Code」→「Download ZIP」→ 解凍

2. Chromeのアドレスバーに `chrome://extensions` と入力して開く

3. 右上の「**デベロッパーモード**」をオンにする

4. 「**パッケージ化されていない拡張機能を読み込む**」をクリックし、解凍したフォルダを選択

## 使い方

1. ニコニコ動画または生放送の再生ページを開く
2. 拡張機能のアイコンをクリックしてポップアップを開く
3. 「録画開始」ボタンを押す
4. 録画を途中で止めたいときは「録画停止」を押す
5. 動画が自動的に保存されます

### キーボードショートカット

| 操作 | Windows / Linux | Mac |
|------|----------------|-----|
| ポップアップを開く | `Ctrl+Shift+Y` | `Cmd+Shift+Y` |
| 録画 開始/停止 | `Ctrl+Shift+R` | `Cmd+Shift+R` |
